# Sgraal VS Code Extension
Validate AI agent memory directly from VS Code.
## Install
Search "Sgraal Memory Governance" in VS Code marketplace.
## Commands
- `Sgraal: Run Preflight on Selection` — select JSON, run preflight
- `Sgraal: Run Preflight on File` — run on entire .jsonl file
## Settings
- `sgraal.apiKey` — API key (default: sg_demo_playground)
- `sgraal.apiUrl` — API URL
- `sgraal.defaultDomain` — Default domain
